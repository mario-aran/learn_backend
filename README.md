# Learn SQL

## Intro

- Concepts: SQL, DB, Tables, Columns
- Tools: RDBMS, Clients
- Syntax: Sentences, `;`
- Comments: `--`, `/* */`

## Selection

- `SELECT`
- Arithmetic Operators: `+`, `-`, `*`, `/`, `%`, `()`
- `FROM`
- Aliases: `AS`
- `DISTINCT`

## Filters

- `WHERE`
- Comparison Operators: `=`, `>`, `<`, `>=`, `<=`, `<>`
- Logical Operators: `NOT`, `AND`, `OR`, `BETWEEN`, `IN`, `LIKE`
- Null Values: `IS NULL`, `IS NOT NULL`
- `CASE`

## Aggregation

- Aggregate Functions: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
- `GROUP BY`
- `HAVING`

## Order

- `ORDER BY`
- By Columns
- By Direction: `ASC`, `DESC`

## Joins

- `(INNER) JOIN`
- `LEFT (OUTER) JOIN`
- `RIGHT (OUTER) JOIN`
- `FULL (OUTER) JOIN`
